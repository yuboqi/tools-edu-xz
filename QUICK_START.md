# 🚀 快速开始指南

## 一分钟上手

### 1. 打开系统

- 双击 `index.html` 文件，或在浏览器中打开它
- 您会看到体育工具系统的主页，显示两个工具卡片

### 2. 选择您需要的工具

#### 选项 A：弯道跑计分系统

1. 点击"弯道跑计分系统"卡片
2. 或直接访问 `index.html?scoring`
3. 开始为比赛打分

#### 选项 B：体测数据对比系统

1. 点击"体测数据对比"卡片
2. 或直接访问 `index.html?compare`
3. 开始编辑和可视化数据

### 3. 使用导航栏

- 页面顶部有导航栏，可以随时切换不同工具
- 点击品牌名称"🏃‍♂️ 体育工具系统"返回主页

## 🎯 快速测试

### 测试弯道跑计分系统

```
1. 访问 index.html?scoring
2. 为每个组别（A、B、C、D）点击不同的名次按钮
3. 点击"提交本轮成绩"
4. 查看总分排行榜更新
5. 点击"📊 导出Excel"测试导出功能
```

### 测试体测数据对比系统

```
1. 访问 index.html?compare
2. 查看默认的示例数据和图表
3. 在表格中修改任意数据
4. 点击"🔄 刷新图表"查看更新
5. 点击"📤 导出图片"测试导出功能
```

## 📱 移动端使用

系统完全支持移动设备：

- 在手机或平板上打开 `index.html`
- 所有功能都可以正常使用
- 界面会自动适配屏幕尺寸

## ⚡ 常见问题

### Q: 数据会丢失吗？

A: 弯道跑计分系统的数据会自动保存到浏览器本地存储，刷新页面不会丢失。

### Q: 可以同时打开多个工具吗？

A: 可以，在新标签页中打开不同的工具即可。

### Q: 导出的文件在哪里？

A: 导出的文件会保存到浏览器的默认下载文件夹。

### Q: 如何重置数据？

A: 在弯道跑计分系统中，点击"重置所有数据"按钮。

### Q: 体测数据对比的图表不显示？

A: 请确保网络连接正常，系统需要加载 ECharts 图表库。

1. **收藏常用工具**: 直接收藏带参数的 URL，下次快速访问
