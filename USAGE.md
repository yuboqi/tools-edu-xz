# 🚀 快速使用指南

## 第一次使用

### 1. 安装 Node.js

如果您还没有安装 Node.js,请先访问 [Node.js 官网](https://nodejs.org/) 下载并安装。

检查是否安装成功:

```bash
node -v
npm -v
```

### 2. 安装项目依赖

在项目根目录下打开终端,运行:

```bash
npm install
```

等待安装完成(可能需要几分钟时间)。

### 3. 启动项目

```bash
npm run dev
```

看到以下提示表示启动成功:

```
ready - started server on 0.0.0.0:3000
```

### 4. 打开浏览器

在浏览器中访问: `http://localhost:3000`

## 功能使用

### 🏃‍♂️ 弯道跑计分系统

**适用场景**: 运动会弯道跑比赛实时计分

1. **开始比赛**

   - 点击主页的"弯道跑计分系统"卡片
   - 系统自动开始第 1 轮比赛

2. **记录成绩**

   - 为每个组别(A、B、C、D)选择名次
   - 点击对应的"第 1 名"、"第 2 名"等按钮
   - 选中后按钮会高亮显示

3. **提交轮次**

   - 确认所有组别都已选择名次
   - 点击"提交本轮成绩"按钮
   - 总分排行榜自动更新

4. **查看结果**

   - 顶部显示实时总分排行榜
   - 底部显示完整的比赛历史记录

5. **导出数据**

   - 总分排行榜右侧有"📊 导出 Excel"按钮
   - 比赛历史右侧有"📋 导出 Excel"按钮
   - 导出的是 CSV 格式,可用 Excel 打开

6. **重置比赛**
   - 新比赛开始前点击"重置所有数据"
   - 会清空所有记录,请谨慎操作

### 📊 体测数据对比系统

**适用场景**: 班级体测数据可视化对比分析

#### 基础操作

1. **创建工作区**

   - 点击右上角的"➕"按钮
   - 输入班级名称,如"2302 班数据对比"
   - 点击确认

2. **切换工作区**

   - 点击顶部的工作区标签
   - 每个工作区独立保存数据

3. **编辑数据**

   - 在表格中直接输入数据
   - 修改项目名称和系列名称
   - 数据会自动更新到图表

4. **保存数据**
   - 点击左上角"💾 保存进度"按钮
   - 数据保存到浏览器本地
   - 右上角显示保存状态

#### 高级功能

1. **调整项目顺序**

   - 按住表格左侧的"☰"图标
   - 拖动到目标位置
   - 松开鼠标完成移动

2. **添加/删除项目**

   - 点击"➕ 添加项目"新增项目
   - 点击行末的"🗑️"删除项目

3. **添加/删除系列**

   - 点击"➕ 添加系列"新增数据系列
   - 在弹窗中输入系列名称
   - 点击表头的"🗑️"删除系列

4. **重命名工作区**

   - 双击工作区标签
   - 输入新名称
   - 按回车或点击确认

5. **删除工作区**
   - 点击工作区标签上的"×"
   - 确认删除(至少保留 1 个工作区)

#### 数据备份与迁移

1. **导出数据**

   - 点击"📤 导出数据"
   - 下载 JSON 文件到电脑
   - 文件包含所有工作区数据

2. **导入数据**

   - 点击"📥 导入数据"
   - 选择之前导出的 JSON 文件
   - 确认导入(会覆盖当前数据)

3. **导出图表图片**
   - 点击"📤 导出图片"
   - 自动下载 PNG 格式图表
   - 可用于报告、演示等

## 常见问题解决

### Q: 启动时报错 "Cannot find module"

**解决方法**: 删除 `node_modules` 文件夹和 `package-lock.json`,重新运行 `npm install`

### Q: 页面样式显示不正常

**解决方法**:

1. 清除浏览器缓存
2. 重启开发服务器 (Ctrl+C 停止,然后 `npm run dev`)

### Q: 数据突然丢失了

**原因**: 可能清除了浏览器缓存或使用了隐私模式

**预防措施**: 定期使用导出功能备份数据

### Q: 图表显示空白

**解决方法**:

1. 检查网络连接
2. 确保表格中有数据
3. 刷新页面重试

### Q: 在其他电脑上使用

**方法 1 - 部署到服务器**:

```bash
npm run build
npm run start
```

**方法 2 - 数据迁移**:

1. 在电脑 A 导出数据(JSON 文件)
2. 传输文件到电脑 B
3. 在电脑 B 导入数据

## 快捷操作

### 键盘快捷键

- `Tab`: 在表格中快速切换输入框
- `Enter`: 在弹窗中确认操作
- `Esc`: (未来版本)关闭弹窗

### 鼠标操作

- **单击**: 选择、编辑
- **双击**: 重命名工作区标签
- **拖拽**: 调整项目顺序

## 最佳实践

### 弯道跑计分

1. **比赛前**:

   - 确认组别数量正确
   - 重置上次比赛数据

2. **比赛中**:

   - 每轮比赛结束立即记录
   - 定期导出备份数据

3. **比赛后**:
   - 导出总分排行榜和历史记录
   - 保存 CSV 文件作为正式记录

### 体测数据对比

1. **工作区命名规范**:

   - 使用统一格式: "XX 班 YYYY 年体测"
   - 例如: "2301 班 2024 年体测"

2. **数据录入**:

   - 先创建所有需要的项目和系列
   - 批量录入数据后统一保存
   - 录入完成后立即导出备份

3. **图表制作**:

   - 调整项目顺序使图表更易读
   - 使用有意义的系列名称
   - 导出高清图片用于报告

4. **数据管理**:
   - 每个班级独立工作区
   - 定期导出 JSON 文件备份
   - 重要数据多处备份

## 技巧提示

💡 **提示 1**: 体测数据对比系统支持拖拽排序,可以快速调整项目展示顺序

💡 **提示 2**: 所有数据都保存在浏览器本地,不会上传到服务器,数据安全有保障

💡 **提示 3**: CSV 文件可以直接用 Excel 打开,方便进一步处理

💡 **提示 4**: 图表支持鼠标滚轮缩放,项目太多时很有用

💡 **提示 5**: 工作区可以随时重命名,不影响数据
